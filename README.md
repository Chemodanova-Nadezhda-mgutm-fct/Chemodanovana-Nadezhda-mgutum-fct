Научная основа
Проект продолжает исследования, проведённые в МГУТУ:
﻿﻿- ВКР «Проектирование программного обеспечения для бизнеса в пищевой промышленности и отраслях агропромышленного комплекса», прикладная информатика, 2025
- ﻿﻿Публикация в РИНЦ: «Новые технологии в индустрии питания - 3D-печать» ([Гришин Александр Сергеевич, Бредихина Ольга Валентиновна, Помоз Алексей Сергеевич, Пономарев Владимир Геннадьевич, Красуля Ольга Николаевна], [2016]) - [[ссылка на elibrary.ru](https://elibrary.ru/item.asp?id=26254150)]
Ключевой метод: это моделирование методом послойного нанесения/наплавления (Fused Deposition Modeling, FDM), применяемый для создания пищевых продуктов из специального сырья путём последовательного нанесения слоёв материала согласно цифровой 3D-модели..

Чек-лист первого контрибьюта:
1. Сделайте форк репозитория (кнопка Fork в правом верхнем углу).
2. ﻿﻿﻿Клонируйте свой форк локально:
   git clone https://github.com/ВAШ-ЛОГИН/ваше-имя-mgutu-fct.git
﻿﻿﻿3. Перейдите в папку проекта:
   cd Chemodanova-Nadezhda-mgutu-fct
4. Установите зависимости:
   pip install -r requirements.txt
5. Запустите тесты:
   python -m pytest tests/
6. Внесите правки → зафиксируйте → отправьте PR.
